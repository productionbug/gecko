# Installation

This guide will help you install and configure HexPacket UI in your project.

## Prerequisites

Before installing HexPacket UI, ensure you have:

- **Node.js** 18.0.0 or higher
- **React** 19.2.0 or higher
- **React DOM** 19.2.0 or higher

## Package Installation

Install HexPacket UI using your preferred package manager:

```bash
npm install @hexpacket/ui
```

```bash
pnpm add @hexpacket/ui
```

```bash
yarn add @hexpacket/ui
```

## Style Setup

### 1. Import Base Styles

Import the main stylesheet in your root layout or main entry file:

```tsx title="app/layout.tsx"
import '@hexpacket/ui/styles.css';

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  );
}
```

```tsx title="src/main.tsx"
import '@hexpacket/ui/styles.css';
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
```

### 2. Import styles for tailwindcss

If you use `tailwindcss`, make sure to import the css inside `layer` directive to correctly override the styles:

```css title="src/global.css"
@import "tailwindcss";

@layer components {
  @import "@hexpacket/ui/styles.css";

  /* Optional if you use Markdown component */
  @import "@hexpacket/ui/markdown.css"; 
}
```

### 3. Import Markdown Styles (Optional)

If you're using the Markdown component, also import its stylesheet:

```tsx
import '@hexpacket/ui/markdown.css';
```

## Container Setup

Add `HexpacketUIContainer` to your layout. This is **required** for the following components:

- Dialog
- Drawer
- ConfirmDialog
- Toast

```tsx title="app/layout.tsx"
import { HexpacketUIContainer } from '@hexpacket/ui';
import '@hexpacket/ui/styles.css';

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>
        {children}
        <HexpacketUIContainer />
      </body>
    </html>
  );
}
```

```tsx title="src/App.tsx"
import { HexpacketUIContainer } from '@hexpacket/ui';

function App() {
  return (
    <div>
      {/* Your app content */}
      <HexpacketUIContainer />
    </div>
  );
}

export default App;
```
