## Quick Start

### 1. Install the package

```bash
npm install @hexpacket/ui
# or
pnpm add @hexpacket/ui
# or
yarn add @hexpacket/ui
```

### 2. Import styles

Add the CSS imports to your root layout or main entry file:

```tsx
import '@hexpacket/ui/styles.css';
```

For markdown component styling, also import:

```tsx
import '@hexpacket/ui/markdown.css';
```

### 3. Add HexpacketUIContainer

Add `HexpacketUIContainer` to your layout (required for Dialog, Drawer, ConfirmDialog, and Toast components):

```tsx
import { HexpacketUIContainer } from '@hexpacket/ui';

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>
        {children}
        <HexpacketUIContainer />
      </body>
    </html>
  );
}
```

### 4. Start using components

```tsx
import { Button, Input, Label } from '@hexpacket/ui';

function LoginForm() {
  return (
    <form>
      <div>
        <Label>Email</Label>
        <Input type="email" placeholder="you@example.com" />
      </div>
      <Button type="submit">Sign In</Button>
    </form>
  );
}
```
