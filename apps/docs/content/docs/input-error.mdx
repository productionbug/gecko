# InputError

A component for displaying error messages below input fields. This is a simple styled div, typically used to show validation errors or other feedback related to form inputs.

## Installation

```tsx
import { InputError } from '@hexpacket/ui';
```

## Basic Usage

<Preview>
  <InputError>This field is required</InputError>
</Preview>

```tsx
<InputError>This field is required</InputError>
```

## Props API

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Error message content |
| `className` | `string` | - | CSS class for custom styling |
| ...rest | `HTMLAttributes<HTMLDivElement>` | - | All standard div attributes |

## Examples

### With Input Field

<Preview>
  <div className="space-y-1">
    <Input
      type="email"
      placeholder="email@example.com"
    />
    <InputError>Please enter a valid email address</InputError>
  </div>
</Preview>

```tsx
import { Input, InputError } from '@hexpacket/ui';

function EmailField() {
  const [error, setError] = useState('');

  return (
    <div className="space-y-1">
      <Input type="email" placeholder="email@example.com" />
      {error && <InputError>{error}</InputError>}
    </div>
  );
}
```

```tsx
  const [errors, setErrors] = useState({});

  return (
    <form>

        {errors.username && (
          <InputError>{errors.username}</InputError>

      </div>

      <div className="space-y-1">
        <Input type="password" placeholder="Password" />
        {errors.password && (
          <InputError>{errors.password}</InputError>
        )}
      </div>
    </form>
  );
}
```

### Multiple Error Types

<Preview>
  <div className="space-y-1">
    <Input
      type="password"
      placeholder="Password"
    />
    <InputError>
      Password must be at least 8 characters and contain at least one uppercase letter, one lowercase letter, and one number
    </InputError>
  </div>
</Preview>

```tsx
function PasswordField({ errors }) {
  const getErrorMessage = () => {
    if (errors.required) return 'Password is required';
    if (errors.minLength) return 'Password must be at least 8 characters';
    if (errors.pattern) return 'Password must contain letters and numbers';
    return '';
  };

  const errorMessage = getErrorMessage();

  return (
    <div className="space-y-1">
      <Input type="password" />
      {errorMessage && <InputError>{errorMessage}</InputError>}
    </div>
  );
}
```

## Accessibility

- Uses semantic HTML (`<div>` element)
- Should be associated with form field via ARIA
- Error styling provides visual feedback
- Works with screen readers

## Styling

The component uses the class name `HPuiInputError`. Override styles using the `className` prop:

```css
.HPuiInputError {
  /* Default error styling */
}
```
